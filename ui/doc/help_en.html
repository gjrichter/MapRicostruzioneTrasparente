how to use

+   2 basic modes

There are two basic modes of interactivity, indicated and changed by the button in the upper left of the map.
In the 'zoom/pan' mode, the mouse may be used to zoom and pan the map like usual with the base map API. 
In the 'info' mode, the mouse is active to show tooltips and information on map and theme elements on mouse over and click.

The various themes of the visualization can be activated by links embedded within the story aside.

  change the representation

The button opens a tool bar to control the map change the representation of the active theme.
The tool bar has various states. By clicking on the button on the left, you go through the various contents.

 select and aggregate

In the selection state there are two tools to define  rectangular or  circular buffers to select theme elements on the map and accumulate the respective values.
These tools act always on the active theme. To activate one theme, in the case we have more than one, you must click on the legend of the theme you want to activate

 share what you see

You can share exactly what you see using the share function. So after you have chosen a theme, may be changed some parameters of its representation, and panned and zoomed the map to the area of your interest, you can use this function to get an URL to reproduce the map anywhere in a HTML5 capable browser.

-  reuse and embed

You can get also a HTML code to embed the map with the actual theme and envelope within an <iframe>.
